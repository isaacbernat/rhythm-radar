How to get the cordova build going

Copy the necessary files from the root of the groovescribe directory into cordova/www

    index.html
    gscribe_help.html
    gscribe_about.html
    js/
    css/
    images/
    MIDI.js/
    soundfont/
    font-awesome/
    
    cordova platform add android
    ...
    
Use the cordova/css/groove_writer_orange.css for now.
